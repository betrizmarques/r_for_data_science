  1. Por que NA ^ 0 não é um valor faltante? Por que NA | TRUE não é um valor faltante? Por que FALSE & NA não é um valor faltante? Você consegue descobrir a regra geral?

Resposta: Pois qualquer número elevado a 0 é 1, mesmo se o NA for TRUE ou FALSE. Pois qualquer coisa OU TRUE é verdadeiro. Pois qualquer coisa & FALSE, dá FALSE pela regra de lógica matemática. A regra geral é que o R não retornará NA se qualquer coisa que substituir aquele NA, dá o mesmo resultado. Exemplo: TRUE | FALSE, dá TRUE, ou seja se o NA for FALSE, TRUE ou até mesmo um número real, ele vai retornar TRUE. Isso aplica até mesmo para NA do tipo NA_character_.
